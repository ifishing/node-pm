node-pm

Run your nodejs service in style (think php-fpm for nodejs)

Installation

Install node-pm

    npm install -g node-pm

Usage

Run application

    node-pm app.js

List node-pm options

    node-pm

Run Tests

    npm test

Generate Docs

Install jsdoc

    npm install -g jsdoc

Generate Docs

    npm run-script docs
