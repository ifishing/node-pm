node-pm [![Build Status](https://travis-ci.org/sazze/node-pm.png?branch=master)](https://travis-ci.org/sazze/node-pm)
====================

Run your nodejs service in style (think php-fpm for nodejs)

Usage
====================

### Run application

``` bash
    node-pm app.js
```

### List node-pm options

``` bash
    node-pm
```

Run Tests
====================

```
  npm test
```
